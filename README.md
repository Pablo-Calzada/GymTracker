# 🏋️‍♂️ GymTracker - Gestión de Entrenamientos
Proyecto desarrollado por **Pablo Calzada**. Una aplicación web completa para la gestión y seguimiento de ejercicios físicos.

## 🚀 Tecnologías Utilizadas
* **Backend:** PHP 8.2 (Arquitectura API REST)
* **Base de Datos:** MySQL (XAMPP)
* **Frontend:** HTML5, CSS3 (Bootstrap 5) y JavaScript Vanilla (Fetch API)
* **Control de Versiones:** Git & GitHub

## 🛠️ Instalación y Configuración
1. Clonar el repositorio en la carpeta `htdocs` de XAMPP.
2. Importar la base de datos desde el archivo `Calzada_Santaeufemia_bd.sql` incluido en el proyecto.
3. Configurar el archivo `.env` con las credenciales locales de la base de datos.
4. Acceder a través de `http://localhost/GymTracker/index.html`.

## 📌 Funcionalidades (CRUD)
- **CREATE:** Añadir nuevos ejercicios al catálogo con validación.
- **READ:** Visualización dinámica de ejercicios mediante consumo de API.
- **UPDATE:** Edición de ejercicios existentes.
- **DELETE:** Eliminación permanente de registros.
